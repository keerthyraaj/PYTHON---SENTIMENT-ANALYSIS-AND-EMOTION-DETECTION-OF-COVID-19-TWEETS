# SENTIMENT ANALYSIS AND EMOTION DETECTION OF COVID-19 TWEETS

# Tools:Python, Pandas, Hugging Face, NLP

Used Python to access Hugging Face, an open-source platform, built-in python, that has pre-trained Natural Language
Processing (NLP) models, and performed Sentiment Analysis and Emotion detection behind the COVID-19 tweets to get to know
the mental state of people during the COVID-19 pandemic.
